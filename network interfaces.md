#Networking
## What are Network Interfaces?
A **network interface** is a connection point that allows a device to communicate over a network.
## Key Points

- Can be **hardware** (NIC card) or **software** (virtual interface).
- Provides a **unique IP address** to the device.
- Used to send and receive network data.
- Each interface has a **MAC address** and may have one or more **IP addresses**.

## Types of Network Interfaces

- **Wired Interface (Ethernet)**
- **Wireless Interface (Wi-Fi)**
- **Loopback Interface (127.0.0.1)**
- **Virtual Interfaces (VMware, Docker, etc.)**
